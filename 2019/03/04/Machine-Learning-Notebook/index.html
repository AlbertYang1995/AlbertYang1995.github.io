决策树

决策树的学习是由训练数据集估计条件概率模型。
决策树学习的损失函数通常是正则化的极大似然函数。决策树学习的策略是以损失函数为目标函数的最小化。
从所有可能的决策树中选取最优决策树是NP完全问题，所以现实中决策树学习算法通常采用启发式方法，近似求解这一最优化问题。
决策树的生成对应于模型的局部选择，决策树的剪枝对应于模型的全局选择。

特征选择

特征的选择通常使用信息增益、信息增益比、基尼系数作为指标。

信息增益

$P(X = x_i) = p_i,i = 1,2,…,n$

熵表示随机变量不确定性的度量，熵越大，随机变量的不确定性就越大。

$H(X) = -\sum_{i=1}^{n}p_ilogp_i$

$Ent(D) = -\sum_{k=1}^{|y|}p_klog_2p_k$

条件熵$H(Y|X)$表示在已知随机变量$X$的条件下随机变量$Y$的不确定性，定义为$X$给定条件下$Y$的条件概率分布的熵对$X$的数学期望。

$H(Y|X) = \sum_{i=1}^{n}p_iH(Y|X = x_i)$

一般的，熵和条件熵之差成为互信息。决策树学习中的信息增益等价于训练数据集中类与特征的互信息。

$Gain(D,a) = Ent(D) - \sum_{v=1}^{V}\dfrac{|D^v|}{|D|}Ent(D^v)$

信息增益大的特征具有更强的分类能力。
